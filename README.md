# 🧁 Manam Bakery Database Management System

A complete **Oracle SQL & PL/SQL based Database Management System** developed as part of the **CSE 302 Database Management Sessional** course.
This project models the internal operations of a bakery business by managing customers, products, inventory, employees, orders, payments, deliveries, and feedback through a structured relational database design.

## 🍰 Project Overview:

Manam Bakery Database System is designed to organize and manage bakery operations efficiently using relational database concepts.

The project focuses on:
- Database design
- Table relationships
- Primary & foreign keys
- Data integrity
- SQL queries
- Oracle PL/SQL programming

The goal was to transform a real-world bakery scenario into a functional database system.

# ✨ Features:

## 👥 Customer Management
- Store customer information
- Maintain customer details
- Manage customer membership records

## 🏪 Branch Management
- Manage bakery branches
- Store branch information and addresses

## 👨‍💼 Employee & Supplier Management
- Maintain employee records
- Manage supplier information

## 🧁 Product & Inventory System
- Manage bakery products
- Organize products using categories
- Track inventory information

## 🛒 Order Management
- Store customer orders
- Maintain order details
- Connect products with customer purchases

## 💳 Payment & Delivery Tracking
- Manage payment information
- Track delivery records

## ⭐ Customer Feedback
- Store and manage customer reviews/feedback

# 🚀 How to Use

1. Open the `.sqlnb` file using an Oracle SQL compatible environment.
2. Execute the scripts in order:

Drop existing tables
Create tables
Insert data
Execute queries
Run PL/SQL programs

3. Ensure Oracle DBMS is properly configured.
